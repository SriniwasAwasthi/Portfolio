# 🚀 Sriniwas Awasthi — Personal Engineering Portfolio

A modern, high-performance personal engineering portfolio built with **Next.js 15**, **TypeScript**, **Tailwind CSS v4**, **Framer Motion**, and **Live GitHub REST API Integration**.

Designed to showcase software engineering craftsmanship, interactive project case studies, and live repository activity.

> 🌐 **Live Website:** [https://sriniwas-portfolio.netlify.app](https://sriniwas-portfolio.netlify.app)  
> 🎓 **Education:** 2nd Year Computer Science Engineering Student at **P.D.A College of Engineering, Kalaburagi**

---

## ✨ Features

- ⚡ **Live GitHub API Auto-Sync**: Automatically fetches and renders public GitHub repositories in real time from [@SriniwasAwasthi](https://github.com/SriniwasAwasthi).
- 🏛️ **9 Featured Projects Suite**: Includes interactive case studies for Meridian Living (E-commerce), Java & DSA Tracker, THINKR AI, Stellar Assault Space Shooter, FocusFlow Ultimate, Book Matrix LMS, CalcAll, and HackForge.
- 🎨 **Modern Neon-Glow Visuals**: Dark-first glassmorphic aesthetic with custom HSL neon design tokens.
- 💻 **Interactive IDE Hero Card**: Code preview card with tabbed developer profiles and tech stack JSON.
- 📊 **GitHub Activity Dashboard**: Interactive contribution heatmap (38+ contributions) and dynamic language distribution breakdown.
- 📱 **100% Responsive & Accessible**: WCAG 2.2 AA compliant with keyboard navigation and semantic HTML.
- 🌐 **Netlify Deployment Ready**: Includes pre-configured `netlify.toml` for zero-configuration Netlify CI/CD deployment.

## 🚀 Quick Start Guide

### Prerequisites

- **Node.js**: `>=18.0.0`
- **npm**: `>=9.0.0`

### Installation & Local Run

```bash
# 1. Clone the repository
git clone https://github.com/SriniwasAwasthi/Protfolio.git
cd Protfolio

# 2. Install dependencies
npm install

# 3. Start local development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) or [http://localhost:3001](http://localhost:3001) in your browser.

## 🌐 Deploying to Netlify (Continuous Deployment)

1. Push your code to GitHub.
2. Log into [Netlify.com](https://www.netlify.com/) using your GitHub account.
3. Click **Add new site** ➔ **Import an existing project** ➔ select **GitHub**.
4. Choose your `Protfolio` repository. Netlify auto-detects `netlify.toml` settings:
   - **Build command**: `npm run build`
   - **Publish directory**: `.next`
5. Click **Deploy Portfolio**.
